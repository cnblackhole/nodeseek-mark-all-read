26.3.31更新：给nodeseek导航栏增加了邮箱的超链接
想法缘由：每次右边显示未读红点都要点进去然后点击已读，再单击ok，太麻烦了，

这个是在原本NodeSeek X脚本插件的基础上增加了一键已读的功能，主界面一键已读无需更多步骤即可消除红点

如何自定义按钮位置：

代码搜索【bottom:-25px;right:-50px】

改这两个就行了，一个上下一个左右
